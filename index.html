Retrocadia v0.1 Experimental Web Build

Please note that the browser based version of Retrocadia uses Web Assembly, which at the time this build was created (2019) is still a fairly new and experimental browser feature.

Not all browsers will be able to load and run the game. Once the game is loaded you may need to click on the title screen before the game will recieve keyboard input. In some browsers the game can appear jittery after first loading, though it does appear to settle down after a few minutes.

Click here to proceed and attempt to load Retrocadia Web into your browser

A stable downloadable version of Retrocadia is available Windows, Mac and Linux. You can get it from itch.io: https://yorkshirekev.itch.io/retrocadia
